📘 Lesson 1: What is Artificial Intelligence?

The foundation of everything we're building

🤔 What is AI?

Artificial Intelligence (AI) is the ability of a computer or machine to think and learn like a human. Instead of just following strict rules, AI can recognise patterns, make decisions, and improve over time.

💡 Simple definition: AI is like teaching a computer to be smart – it can see, hear, understand, and even create things!

📊 The 3 Types of AI

1. Narrow AI (Weak AI)

What it does: One specific task really well.
Examples: Voice assistants (Siri, Alexa), Netflix recommendations, facial recognition, self-driving cars.
Where we are: This is where we are right now!

2. General AI (Strong AI)

What it does: Any intellectual task that a human can do.
Examples: A machine that can write poetry, code software, cook a meal, and have a conversation – all by itself.
Where we are: Still being researched (we're not there yet).

3. Super AI

What it does: Smarter than the smartest human in every way.
Examples: A machine that can solve problems humans can't even imagine.
Where we are: Science fiction (for now!).

⚙️ How Does AI Work?

AI works through Machine Learning – which is like teaching a child, but with data instead of toys.

The 4-step process:

  • 1. Data Collection: The AI is given lots of information (images, text, numbers).
  • 2. Training: The AI looks for patterns in the data.
  • 3. Testing: The AI is tested to see if it learned correctly.
  • 4. Improvement: The AI gets better over time with more data.

🧠 Example: To teach an AI to recognise cats, you show it 10,000 cat photos. It learns that cats have ears, whiskers, and tails. Then it can spot a cat in a new photo!

// Your starfield code
function drawStars() {
    ctx.clearRect(0, 0, width, height);

    for (let star of stars) {
        ctx.beginPath();
        ctx.arc(star.x, star.y, star.radius, 0, Math.PI * 2);
        ctx.fillStyle = 'rgba(255, 255, 255, ' + star.opacity + ')';
        ctx.fill();
        star.x += star.speed * star.direction; // Move sideways
    }

    requestAnimationFrame(drawStars); // Keep it running
}
