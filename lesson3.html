🌐 Lesson 3: Building Your First AI Website

Take your website from static to smart with AI

🤔 What is an AI-Powered Website?

An AI-powered website is a normal website that can think, learn, and respond like a human. Instead of just showing static information, it can talk to users, answer questions, and even generate content.

💡 Real-world example: You just built one! Your chat box and "Generate Instructions" button are powered by AI.

🔧 Step 1: Choose Your AI Provider

There are many AI providers you can use:

  • OpenAI – The most popular (ChatGPT, GPT-4)
  • Mistral AI – Open-source and powerful
  • Google Gemini – Google's AI
  • Claude – Anthropic's AI
  • Local LLM – Run on your own computer (free!)

📡 Step 2: Connect to an AI API

An API is like a phone line between your website and the AI. Your website sends a message, the AI thinks about it, and sends a response back.

// How to call an AI API async function getAIResponse(userMessage) {
    const response = await fetch('https://api.ai-provider.com/v1/chat', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            messages: [{ role: 'user', content: userMessage }]
        })
    });
    return await response.json();
}

📝 Step 3: Write Your AI Logic

You already did this in your chat.js file! Your code:

  • Takes the user's message
  • Sends it to the AI
  • Shows the AI's response in the chat box
  • Handles errors if the AI is busy

🚀 Key insight: The AI doesn't know anything about your website. You have to tell it what your website is about using the system message.

🎨 Step 4: Design Your UI

An AI website needs a good user interface. You've already built:

  • Chat box – Where users type messages
  • Message bubbles – User messages (right) and AI responses (left)
  • Loading states – "AI is thinking..." indicator
  • Action buttons – "Generate Instructions" and "Copy" buttons

⚡ Step 5: Test and Iterate

  • Test different prompts: Try asking your AI different questions
  • Handle edge cases: What if the AI is slow? What if it fails?
  • Improve responses: Update the system message to get better answers
